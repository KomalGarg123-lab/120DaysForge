This repo tracks my 4-month learning journey from **1st April to 31st July**. Each day, I will update what I completed in ML, DL, Full-Stack projects, and DSA practice. The goal is to revise, learn, build projects, and improve problem-solving consistently.

1. **Machine Learning (ML)**  
   - Target: Revise ML concepts thoroughly

2. **Deep Learning (DL)**  
   - Target: Learn Deep Learning at a good level  
   - Deploy Deep Learning projects  

3. **Full-Stack Development**  
   - Target: Develop at least 4 big real-life projects  
   - Learn new technologies and concepts while building  

4. **Data Structures & Algorithms (DSA)**  
   - Target: Become very comfortable with medium-level questions
